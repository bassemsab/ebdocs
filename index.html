VLANs

  • VLAN 995: Public Area for Ruckus APs setup (Levels 4,16-28).

    ZD controller IP for VLAN 995 10.100.100.2/10.100.100.5

  • VLAN 990: Public Area for Ruckus APs setup (Levels B2,LL,3,5-17)

  • VLAN 190: GF (Lobby) Printer Setup VLAN at public area.

    ZD controller IP for VLAN 990 and 190: 10.99.99.2/10.99.99.5

  • VLAN 999: Public Room Level, Ruckus AP setup.

  • VLAN 72: Testing VLAN (Avaliable at IT office at LVLB2)

  • VLAN 991: Internet access (ZD controller 10.100.100.2 / 10.100.100.5)

  • VLAN XX99: Floor VLAN, where XX is the floor number.

    Example: VLAN 2099 is for floor 20.

[!NOTE] ***Ruckus last IP take from Horizon Lounge area including switch setting with native vlan 990.

User name: slkl
